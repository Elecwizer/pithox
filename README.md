# PITHOX — 3D Vertical Slice

**PITHOX** is a fast-paced, arena-based survival game focused on risk-reward gameplay and session-based progression.  
This project is a **3D vertical slice prototype** developed in Unity, showcasing the core gameplay loop and systems.

---

## Overview

- **Genre:** Action / Survival Arena  
- **Platform:** PC (Windows)  
- **Engine:** Unity  
- **Development Team:** Team 7  

PITHOX challenges players to survive escalating waves of enemies while balancing combat efficiency and strategic decision-making through a unique offering system.

---

## Core Gameplay Loop

Fight → Collect → Risk → Upgrade → Repeat

1. Defeat enemies in a confined arena
2. Enemies drop **tombstones**
3. Carry tombstones (with movement penalty)
4. Deliver them to **PITHOX**
5. Gain rewards / upgrades
6. Survive as long as possible

---

## Game Objective

- Survive enemy waves as long as possible
- Maximize score and combo
- Strategically decide when to fight vs. when to deliver offerings

---

## ⚙️ Features (Current Vertical Slice)

- Responsive player movement and combat
- Enemy AI with multiple behavior states
- Risk-reward tombstone mechanic
- Combo and scoring system
- Basic UI (HUD, results, defeat screen)
- Audio feedback (music and SFX)

---

## Known Limitations

This is a **prototype / vertical slice**, so some features are incomplete:

- Limited enemy variety
- No full upgrade system (yet)
- No boss fight (yet)
- UI is not fully polished
- Audio is first-pass only
- No settings/options menu
